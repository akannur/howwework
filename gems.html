Special Gems

Gems are packages similar to Ebuilds. They contain package information along with files to install.Gems are usually built from ".gemspec" files, which are YAML files containing information on Gems. However, Ruby code may also build Gems directly. Such a practice is usually used with Rake.

  • ERD gem - is a gem that allows you to easily generate a diagram based on your application's Active Record models. The diagram gives an overview of how your models are related. Having a diagram that describes your models is perfect documentation for your application.
  • JWT gem - JSON Web Token implementation in Ruby.
